Iris Flower Classification

This project classifies Iris flower species using Machine Learning.

Dataset
The dataset contains measurements of iris flowers:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
 
Steps Performed
1. Imported dataset
2. Data exploration
3. Data visualization
4. Train-test split
5. Model training using Random Forest
6. Model evaluation

Result
The model predicts the species of iris flowers with good accuracy.
